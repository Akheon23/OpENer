Returns current time in microseconds from monotonic time base, please note that this does not represent a real absolute time, but measured from an arbitrary starting point.

This function returns the current time relative to an arbitrary starting point from a monotonic time source. As monotonic clocks and clock functions in general are platform dependent, this has to be implemented for each platform (see ports subfolders)
